Le mie esperienze lavorative

  • Linguaggi di programmazione usati con frequenza: Python, Node.js, JavaScript, TypeScript
  • Linguaggi con conoscenza base / da rispolverare: Java, PHP, C++, C
  • Linguaggi per lo sviluppo web: HTML, CSS, Vue.js
  • Gestione di database: sempre usato Postgres in ambito lavorativo, mentre a scuola MySQL
  • Framework Odoo: sviluppo di moduli / creazione API
  • DevOps & ambienti di lavoro: Docker (conoscenza base), sistemi Linux, controllo versione con Git

Per quanto riguarda le esperienze lavorative, sono principalmente 2: BNova e Sunlink. Qua sotto metto una breve descrizione delle esperienze.

BNova Srl
Qua ho fatto la mia prima esperienza lavorativa, subito dopo il diploma.
Sono stato qua per 2 anni in totale, da Ottobre 2018 a Ottobre 2020. I primi 6 mesi sono stati di stage e poi sono stato assunto con un contratto di apprendistato. Sono rimasto finché non ho deciso di inseguire i miei sogni (fallimentari) di Game Developer.
BNova è un'azienda che si occupa principalmente di analisi dei dati e della loro rappresentazione. In azienda io non mi occupavo proprio di questo, ma più della parte "tecnica": sviluppo di applicazioni web, risoluzione di bug, test, documentazione e manuali d'uso di piccoli applicativi o di installazioni.
Grazie a questa esperienza ho sviluppato molte competenze in ambito informatico e, più in generale, sul lavorare in gruppo, gestirmi il carico di lavoro e rispettare le scadenze.
In breve le attività / caratteristiche del lavoro svolto:
  • Sviluppo front-end di applicazioni web (HTML, CSS, JavaScript, JQuery).
  • Sviluppo back-end di applicazioni web (linguaggio: Java; server: Apache Tomcat; database: PostgreSQL) su server Linux.
  • Identificazione dei problemi e bug fixing di errori riscontrati durante i test (principalmente di applicazioni web su server Linux).
  • Realizzazione di documenti tecnici per l'installazione di plug-in su applicazioni web.
  • Collaborazione alla redazione dei manuali d'uso delle applicazioni.
  • Conduzione di test (anche automatici), con relativa documentazione, per software di vario genere, applicazioni e siti web.
  • Sviluppo di software in Python con relativa redazione di documenti tecnici per descriverne il funzionamento.

Sunlink Srl
Dopo il fallimento nel tentativo di diventare ricco e vivere di rendita per sempre tramite la pubblicazione di un videogioco, sono andato a lavorare qua, a Sunlink. In totale ho passato 3 anni qua, da Febbraio 2023 a Marzo 2026.
Sunlink è un'azienda che si occupa di monitoraggio ed efficientamento energetico. In azienda io mi occupavo principalmente dello sviluppo della parte back-end della web app.
In breve le attività / caratteristiche del lavoro svolto:
  • Sviluppo back-end: utilizzo del framework Odoo per la gestione del database (Postgres), sviluppo di API in Python per la web app e altri servizi.
  • Sviluppo software in Node.js sui mini-PC installati nelle sedi monitorate per l'organizzazione, l'elaborazione e la trasformazione di grandi quantità di dati (tramite TimescaleDB).
  • Integrazione di sistemi eterogenei per la raccolta dati da dispositivi industriali tramite i protocolli OPC UA e modbus TCP/RTU (per PLC e meter) e servizi di terze parti tramite API (per inverter GoodWe, ZCS, ABB).
  • Sviluppo test automatici in Python e Node.js.
  • Supporto occasionale allo sviluppo front-end con Vue.js e TypeScript.
